## 使用说明
- 程序会尝试加载默认文件 `list.xlsx`
- 如果未找到默认文件，请点击“文件 -> 导入”选择名单文件
- 可以选择是否允许重复点名
- 点击“点名”按钮进行随机点名
- 点击“重置”按钮清空已点名名单
- 点击“文件 -> 已点名名单”查看已点名名单
